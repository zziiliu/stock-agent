排查发现，每一个agent自己的ReAct循环出现问题。langgraph 中默认是 25 步。通过打印调用工具流程日志，出现 recursion limit 错误是因为两个主要的原因：第一是转换内容为 md 文档的工具没下载，所以会报 error；第二是由于 agent 本身一次会调用多个工具，但是这些工具可能触发 baostock 的并发登录限制，导致某一些工具无法 login。而提示词中又说“需要使用不同工具进行组合”，因此导致 agent 本身触发限制<br>
